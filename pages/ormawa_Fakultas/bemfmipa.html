Badan Eksekutif Mahasiswa Fakultas Matematika dan Ilmu Pengetahuan Alam

Fakultas Matematika dan Ilmu Pengetahuan Alam didirikan pada tanggal 1 Juli 1984. Fakultas Matematika dan Ilmu Pengetahuan Alam memiliki enam program studi yaitu Program Studi Kimia, Program Studi Biologi, Program Studi Matematika, Program Studi Fisika, Program Studi Farmasi, dan Program Studi Teknik Informatika
